1) A cosa serve bootstrap?

Bootstrap serve per creare pagine web con tutti i template di HTML e CSS, e fornisce l'abilità di creare dei desgin responsive in modo veloce.

2) Cosa devi importare per creare una pagina web con bootstrap?

Per creare una pagina web con bootstrap è necessario aggiungere il DOCTYPE di HTML 5, le seguenti scritture nel tag head:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.2.1/css/bootstrap.min.css">>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.6/umd/popper.min.js"></script>
    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.2.1/js/bootstrap.min.js"></script>

e infine i conteiner che vengono utilizzati per contenere la pagina web.

3) Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Il CDN è un mezzo per gli utenti per scaricare i file in un tempo più ristretto perchè prende il file richiesto dal server più vicino.

4) A cosa serve questo attributo del tag meta?

a. content="width=device-width, initial-scale=1

Il seguente tag meta viene utilizzato per adattare il sito web anche a devices più piccoli.

5) Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

Il container normale non occupa tutta la pagina ed è responsive mentre il container fluid occupa tutta la pagina e si espande per tutto lo schermo.
